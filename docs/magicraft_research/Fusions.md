## Overview

Magicraft's spell fusion system represents its core innovation, allowing players to combine different spells and elements to create unique effects with "unexpected amazing effects." The system encourages experimentation and discovery, creating emergent gameplay through spell combination mechanics.

## Fusion Mechanics

### **Fusion Process**
1. **Selection**: Choose two or more spells to fuse
2. **Compatibility Check**: Verify fusion is possible
3. **Resource Cost**: Calculate mana cost for fusion
4. **Cast Time**: Determine fusion casting time
5. **Effect Generation**: Create combined spell effects
6. **Result Application**: Apply fused spell to targets

### **Fusion Cost Calculation**
```
Base Cost = Spell1 Cost + Spell2 Cost
Fusion Multiplier = 1.5 (basic), 2.0 (advanced), 2.5 (ultimate)
Final Cost = Base Cost √ó Fusion Multiplier
Mastery Reduction = -5% per fusion mastery level
```

### **Fusion Success Rate**
- **Basic Fusions**: 100% success rate
- **Advanced Fusions**: 85% success rate
- **Ultimate Fusions**: 70% success rate
- **Mastery Bonus**: +3% per fusion mastery level

```cpp
FusionResult attemptFusion(const vector<Spell>& spells) {
    FusionResult result;
    
    // Check compatibility
    if (!areCompatible(spells)) {
        result.success = false;
        return result;
    }
    
    // Calculate success rate
    float successChance = calculateSuccessRate(spells);
    if (rollSuccess(successChance)) {
        result.fusedSpell = createFusedSpell(spells);
        result.success = true;

    }
```
