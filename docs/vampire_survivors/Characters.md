# Vampire Survivors: Characters Reference

## Overview

Vampire Survivors features **165+ playable characters** across the base game and DLCs. Each character has:
- **Starting Weapon**: Determines initial playstyle
- **Passive Bonus**: Unique stat scaling per level
- **Unlock Condition**: Achievement, gold purchase, or secret code

## Unlock Priority Guide

### Early Game (0-10,000 Gold)

1. **Imelda** (10 Gold) - XP bonus for faster leveling
2. **Pasqualina** (100 Gold) - Runetracer is strong early
3. **Gennaro** (500 Gold) - Extra projectiles scale well
4. **Arca** (500 Gold) - Fire Wand is powerful
5. **Porta** (500 Gold) - Area bonus helps clear

### Mid Game (10,000-50,000 Gold)

1. **Pugnala** (1,000 Gold) - Dual weapons, massive Might scaling
2. **Giovanna** (1,000 Gold) - Gatti Amari is unique
3. **Poppea** (1,000 Gold) - Duration bonus is universally good
4. **Mortaccio** (500 Gold) - Bone scales infinitely

### Late Game (Secrets)

1. **Exdash** - Type code, unlock Toastie path
2. **Toastie** - Extreme challenge character
3. **Queen Sigma** - Requires full collection
4. **Red Death** - Requires killing Death

## Character Cost Scaling

Character costs increase by **10%** after each purchase:
- First character: Base cost
- Second character: Base cost × 1.1
- Third character: Base cost × 1.21
- And so on...

**Tip**: Buy cheaper characters first to minimize total gold spent.
