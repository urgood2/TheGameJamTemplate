# Build Archetypes — Structured Analysis (POA Ultrawork Style)

Format per build: **Core Loop**, **Key Components**, **Stat Priorities**, **Weaknesses**.

---

## Homing Serpent
- **Core Loop**: High-tick homing projectile stacks Poison; sustain via hits.
- **Key Components**: Track + Shadow Serpent + Venom Crystal + Mana Absorption; Swiftcaster wand (machine gun).
- **Stat Priorities**: Cast interval ↓, Regen ↑, Scatter mitigation (Track).
- **Weaknesses**: Low burst; needs space to tick.

## Grimoire Hive
- **Core Loop**: Spawn books → fuse → autonomous firing army.
- **Key Components**: Autonomous Grimoire ×2 + Fusion Summon + Energy Saving + Indomitability; any regen wand.
- **Stat Priorities**: Regen, Summon cost reduction (Novice’s Grimoire), survivability for summons.
- **Weaknesses**: Setup time; protect summons.

## Laser Reflector
- **Core Loop**: Beams bounce/reflect to fill room; apply Venom ticks.
- **Key Components**: Laser/Ray of Disintegration + Reflection/Ricochet + Venom Crystal; wand with decent MP and scatter control.
- **Stat Priorities**: MP regen, scatter mitigation, sustain (Absorption optional).
- **Weaknesses**: Geometry dependent; MP hungry.

## Post-Slot Burst
- **Core Loop**: Charge post-slot via casts/hits; detonate payload when threshold reached.
- **Key Components**: 777/Frenzy/Rusty Blaster/Book of Tranquility + Meteor/Star Arrow in post-slot + Fuse alignment.
- **Stat Priorities**: Charge rate (hit/cast frequency), MP regen, survivability during charge.
- **Weaknesses**: Timing-sensitive; vulnerable windows while charging.

## Volley Payload
- **Core Loop**: Parallel fire multiple payloads with cost discount.
- **Key Components**: Volley + Echo + Area Boost + Arcane Nova/Star Arrow; Stellar Drift/Harmonic Resonance.
- **Stat Priorities**: MP pool/regen, homing (Track), scatter mitigation.
- **Weaknesses**: Mana crash risk; whiff risk without homing.

## Fusion Summoning Army
- **Core Loop**: Maintain pairs of summons → fuse → stat-doubled units.
- **Key Components**: Pop/Grimoire/Skull + Fusion Summon + Indomitability + Energy Saving.
- **Stat Priorities**: Summon cost reduction, regen, defensive buffers.
- **Weaknesses**: If summons die, DPS vanishes; needs Indomitability.

## Area Boost Mimicry
- **Core Loop**: Stack area/damage passives via Mimicry; drop oversized payloads.
- **Key Components**: Area Boost + Mimicry Cube + Meteor/Nova; volley-capable wand preferred.
- **Stat Priorities**: MP pool, regen, modifiers adjacency.
- **Weaknesses**: Setup time; costs if no sustain.
